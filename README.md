# 📁 Creating Personal Google Drive

A simple Node.js project for creating and managing personal Google Drive-style storage locally.

---

## 🚀 Features

- 📂 File Storage System
- 🖼️ Image Folder Support
- 📄 PDF & Text File Management
- ⚡ Node.js Backend
- 🌐 HTML Boilerplate Included
